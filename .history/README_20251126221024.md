## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, or pnpm

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/DiwanMalla/portfolio_macOS.git
cd portfolio_macOS
```

2. **Install dependencies**

```bash
npm install
```

3. **Set up environment variables**

```bash
cp .env.example .env.local
```

Add your API keys:

```env
# Email (Nodemailer)
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password

# AI Chatbot
GROQ_API_KEY=your-groq-api-key
UPSTASH_VECTOR_REST_URL=your-upstash-url
UPSTASH_VECTOR_REST_TOKEN=your-upstash-token
```

4. **Run the development server**

```bash
npm run dev
```

5. **Open in browser**

```
http://localhost:3000
```

## ğŸ¨ Customization

### Change Profile Data

Edit `constants/profile.ts` to update your personal information.

### Add New Windows/Apps

1. Create component in `components/windows/`
2. Register in `constants/apps.ts`
3. Add case in `components/Window.tsx`

### Change Wallpapers

Add images to `public/wallpapers/` and register in `constants/wallpapers.ts`
